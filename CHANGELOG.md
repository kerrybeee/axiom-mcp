# Changelog

## v2.0.0

Initial public release of AXIOM MCP.

Highlights:
- SQLite-backed reasoning ledger for MCP agents
- drift detection and receipt replay
- search, compare, export, and postmortem tools
- browser dashboard, gallery, reports, and arena pages
- persistent policy system
- guard evaluation and next-step recommendation
- branch suggestion and historical learning
- action gating, intervention, and resume flow
